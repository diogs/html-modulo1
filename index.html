H1 até H6

Estas são Tags HTML que indicam os níveis de cabeçalho de uma página web. O H1 é o mais importante e o H6 é o menos importante.

Por exemplo:

Este é o título principal (h1)

Este é um subtítulo (h2)

Este é outro subtítulo (h3)

Este é um sub-subtítulo (h4)

Este é um sub-sub-subtítulo (h5)
Este é um sub-sub-sub-subtítulo (h6)

Tag p

A tag p é usada para criar páragrafos em HTML.

Ela define o início e o fim de um bloco de texto que pode ter formatação,alinhamento e espaçamento.

A tag p é uma tag de conteúdo, ou seja, ela pode conter outros elementos HTML dentro dela.

Tag small

A tag small é um elemento HTML que representa comentários secundários e texto pequeno, como direitos autorais e texto legal, independentemente da sua apresentação estilizada.

Por padrão, ela renderiza o texto dentro dela um tamanho de fonte menor.

Veja um exemplo de uso da tag small:

Este é um texto normal.

Este é um texto menor.

Tag blockquote

A tag blockquote é um elemento HTML que serve para indicar que o texto é uma citação.

Normalmente, os navegadores fazem um recuo visual nos elementos blockquote.

Por exemplo:

Esta é uma citação

Tag abbr

A tag abbr é usada para marcar abreviações ou siglas, como HTML, CSS, ONU, etc.

Ela tem um atributo title que mostra a descrição completa da abreviação quando você passa o mouse sobre ela.

Por exemplo:

ONU

Essa tag ajuda os navegadores, os leitores de tela, os buscadores e os tradutores a entenderem melhor o seu texto.

No entanto, a tag <font> está obsoleta e não é mais suportada no HTML5. O recomendado é usar CSS para estilizar o texto de forma mais eficiente e organizada.

Tag de listas ol,ul e li

Uma lista é uma coleção de itens que são organizados em uma ordem específica ou sem ordem. Existem dois tipos principais de listas:

listas ordenadas e listas não ordenadas.

Listas ordenadas usam números ou letras para marcar cada item, enquanto listas não ordenadas usam símbolos como pontos ou traços.

Para criar uma lista em HTML, você precisa usar as tags <ul> ou <ol> para definir o tipo de lista e as tags <li> para definir cada item da lista.

Por exemplo:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
  • Item sem ordem
  • Outro item sem ordem
  • Mais um item sem ordem

Tag a

A tag <a> é um elemento HTML que serve para criar links entre páginas web, arquivos, e-mails, seções da mesma página ou outras URLs.

O atributo mais importante da tag <a> é o href, que indica o destino do link.

O conteúdo dentro da tag <a> deve indicar o propósito do link.

Por exemplo:

Conheça a DIO e Desbloqueie o seu próximo nível de conhecimento
